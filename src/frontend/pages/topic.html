I've been really into online learning recently. Part of the reason is because I don't think that the high-quality online classes will be free forever. Here are a few of the big players in the arena:

Coursera:
https://www.coursera.org

Udacity:
https://www.udacity.com

edX:
https://www.edx.org

I've taken two classes through edX. They are:

Introduction to Computer Science and Programming:
https://www.edx.org/courses/MITx/6.00x/2013_Spring/about

and Artificial Intelligence:
https://www.edx.org/courses/BerkeleyX/CS188.1x/2013_Spring/about

I mainly signed up for the Intro class because I wanted to learn python. The AI class was taught using python. Since I already know how to program in several other languages, I found that I didn't need the Intro class at all, so I eventually dropped it. The lectures were good, however, and I think it would be a great class for somebody just starting out.

The AI class was taught by UC Berkeley professors and was excellent. I highly recommend it. Word is that the 2nd part of it will be taught next fall -- I'll definitely sign up for that.
